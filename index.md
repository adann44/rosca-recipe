# Receta de Rosca

## Rosca (Masa del Pan)

| Ingrediente | Cantidad |
|-------------|----------|
| Harina | 6 ½ tazas (1,000 g) |
| Huevos (a temperatura ambiente) | 3 |
| Leche entera | 1 2/3 tazas |
| Azúcar | 1 ¼ tazas |
| Mantequilla sin sal | 1 barra + 2 ½ cucharadas |
| Levadura seca (SAF-instant premium red) | 13 g |
| Sal | Pizca |
| Extracto de naranja | 2 cucharaditas |
| Cobertura de azúcar | Al gusto |
| Huevos para barnizar | 2 |

---

## Pasta de Manteca

| Ingrediente | Cantidad |
|-------------|----------|
| Azúcar regular (granulada) | ½ kilo (500 g) ≈ 2½ tazas |
| Azúcar glass | ½ kilo (500 g) ≈ 4¼ tazas* |
| Manteca vegetal Crisco | 1 kilo (1000 g) ≈ 5 tazas + 3 cucharadas + 1 cucharadita |
| Harina | Pizca (para unir) |

*El azúcar glass varía mucho dependiendo del cernido y compactado, por lo que esto es un estimado.
