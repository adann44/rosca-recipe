# Rosca Recipe

## Rosca (Bread Dough)

| Ingredient | Amount |
|------------|--------|
| Flour | 6 ½ cups (1,000 g) |
| Eggs (room temperature) | 3 |
| Whole Milk | 1 2/3 cups |
| Sugar | 1 ¼ cups |
| Unsalted Butter | 1 stick + 2 ½ tbsp |
| Dry Yeast (SAF-instant premium red) | 13 g |
| Salt | Pinch |
| Orange Extract | 2 tsp |
| Sugar coat on toppings | As needed |

## Pasta de Manteca

| Ingredient | Amount |
|------------|--------|
| Regular (granulated) sugar | ½ kilo (500 g) ≈ 2½ cups |
| Powdered sugar | ½ kilo (500 g) ≈ 4¼ cups* |
| Crisco (tub shortening) | 1 kilo (1000 g) ≈ 5 cups + 3 Tbsp + 1 tsp |
| Flour | Pinches (to hold it together) |

*Powdered sugar varies a lot depending on sifting/packing, so this is an estimate.
